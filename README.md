# KIRBYSOFT
Control system for Kirky's house

## Pumps
* **PUMP1** 
  * NORMAL to FAU
  * HIGH to FAU
* **PUMP2** 
  * NORMAL to exchanger
* **PUMP3** 
  * NORMAL to slab
  * HIGH  to slab

## Temperature Sensors
* TEMP1 *Panel supply*
* TEMP2 *East panel return*
* TEMP3 *West panel return*
* TEMP4 *H2O supply*
* TEMP5 *H2O return*
* TEMP6 *H2O main*
* TEMP7 *FAU supply*
* TEMP8 *FAU return*

## Flow Sensors
* FLOW1 *Panel supply*
* FLOW2 *East panel return*
* FLOW3 *West panel return*
* FLOW4 *FAU heat exchanger*
* FLOW5 *H2O heat exchanger*
* FLOW6 *Driveway slab*
